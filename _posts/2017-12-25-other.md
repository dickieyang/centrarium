### 基本命令
- git status
- git init<br>
    输入git init命令，初始化 Git 仓库
- git add<br>
    输入git add hit.txt命令，将hit.txt文件添加到 Git 仓库
- git commit<br>
    输入git commit -m "text commit"命令，将hit.txt文件提交到 Git 仓库
- git log
    
    输入git log"命令，打印 Git 仓库提交日志
